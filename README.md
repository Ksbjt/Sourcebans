# Sourcebans

#### 运行要求

Web端需求：

- MySQL/[MariaDB](https://sbpp.dev/docs/mariadb) 数据库
- PHP 版本 >= 5.6 （建议5.6）

游戏服务器端

- Sourcemod >= 1.7
- Metamod: Source
